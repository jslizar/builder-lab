# n8n Workflows

n8n workflow JSON files ready to import.

## Contents

| Workflow | Description |
|----------|-------------|
| `ai-personalized-outreach-engine.json` | AI-powered SMS outreach with cohort routing, company research via Firecrawl, and personalized message generation using Claude |
| `lead-generation-enrichment.json` | Google Places scraper with Firecrawl website enrichment and OpenCorporates owner lookup. Outputs to Google Sheets |
| `lead-pipeline-v3.json` | Full lead pipeline: multi-source scraping (Google Maps, Yelp, Airbnb), AI validation, enrichment waterfall (Apollo, PDL, BatchData), AI scoring, competitive intel, multichannel outreach, CRM updates, and error handling |
| `social-content-intelligence.json` | Weekly content intelligence report. Pulls posts from 5 platforms, analyzes top performers with Claude, researches trends across Perplexity/Reddit/X/TikTok/Google Trends, generates content ideas, publishes to Notion |
| `sms-outreach-property-managers.json` | SMS outreach engine for property managers with 3 offer tracks ($500 AI Agent, DFY Automations, Consulting). Includes phone validation, AI message writing with website research, and OpenPhone delivery |
| `sms-outreach-service-providers.json` | SMS outreach engine for home service companies (HVAC, plumbing, electrical) with 4 offer tracks. Includes phone validation via Trestle, AI message writing with trade-specific hooks, and send tracking |

## How to Import

1. Open your n8n instance
2. Click **Add workflow** > **Import from file**
3. Select the `.json` file
4. Update all credential nodes with your own API keys
5. Update any hardcoded IDs (Google Sheets, Notion databases, etc.)

## Before You Run

These workflows have been sanitized for sharing. You **must** replace placeholder values before running:

- `YOUR_GOOGLE_PLACES_API_KEY` — Get from Google Cloud Console
- `YOUR_VERIPHONE_API_KEY` — Get from veriphone.io
- `YOUR_TRESTLE_API_KEY` — Get from trestleiq.com
- `+1XXXXXXXXXX` / `+1YYYYYYYYYY` — Your OpenPhone numbers
- `YOUR_GOOGLE_SHEET_ID` — Your Google Sheets document ID
- `YOUR_WEBHOOK_PATH_UUID` — Auto-generated by n8n on import
- All credential nodes need to be reconnected to your own credentials
